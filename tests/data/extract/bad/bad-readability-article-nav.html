Migration Guide: Jobs v2

Jobs v2 introduces a new retry model. Follow the steps below to migrate without downtime.

Step 1: Update the client

pip install acme-jobs==2.0.0

Step 2: Update the config

{
  "jobs": {
    "retries": 5,
    "backoff": "exponential"
  }
}

Step 3: Verify

Run a dry-run and check the dashboard for failures.
