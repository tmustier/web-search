# Security

This project fetches and processes untrusted web content. Treat tool outputs as untrusted input.

## Reporting a vulnerability

If you believe you’ve found a security issue:

- Prefer using GitHub’s private vulnerability reporting (Security Advisories) if enabled for this repo.
- Otherwise, open an issue with **no sensitive details** and ask for a private channel.

Please do not include secrets (API keys, cookies, access tokens) in reports.
